🏆 Game Roles & Requirements

Developer

To become a certified game developer, you must submit an application, which, currently is not sat up.

Requirements:
  • 14+ years of age
  • 6+ months of web/game development
  • Flexible schedule (Not strict at all)
  • Can easily work in teams
  • Can be self-motivated

Programming languages:
  • HTML, and CSS
  • JavaScript
  • TypeScript
  • NextJS (Serverless backend)

Admin

To become a platform admin, you must submit an application, which, currently is not sat up.

Requirements:
  • Must be a promoted moderator

Moderator

To become a moderator, you must submit an application, which, currently is not sat up.

Requirements:
  • Must be a promoted from Helper

Helper

To become a platform admin, you must submit an application, which, currently is not sat up.

Requirements:
  • Must not be a passed banned member
